# Finteck — Modern Finance Landing Page

A sleek, modern fintech landing page built with **Next.js, Tailwind CSS, and Framer Motion**.

Designed with a strong focus on:
- ✨ clean UI
- 🎯 motion hierarchy
- ⚡ smooth scrolling experience
- 🎨 minimal & premium design

![Hero Section](./public/image.png)

## 🛠️ Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Lenis (smooth scrolling)

---

## 🎬 Features

- Smooth scroll-based experience (Lenis)
- Section-based motion system
- Clean stagger animations
- Responsive layout (mobile-first)
- Modern fintech UI design
